# SpeakUp — Brand Guidelines

## Colours

| Token | Hex | Usage |
|-------|-----|-------|
| Background | #0A0F1E | Deep navy — all screens |
| Primary | #00E5CC | Electric teal — non-church decks, primary actions |
| Prayer accent | #C9922A | Warm gold — church-prayer, church-announcements |
| Success | #22C55E | Green — "Nailed it" result |
| Warning | #F59E0B | Amber — "Close" result |
| Error | #EF4444 | Red — "Try again" result |
| Text primary | #F1F5F9 | Light grey — headings, body |
| Text muted | #94A3B8 | Dimmer grey — secondary text |

## Typography

| Role | Font | Weight |
|------|------|--------|
| Heading | DM Serif Display | 400 |
| Body | DM Sans | 400, 500, 700 |

## Spacing & Sizing

- Min tap target: 48px
- MicButton minimum: 80px diameter
- Mobile-first: 375px minimum width
- Border radius: 12px–24px (rounded-xl / rounded-2xl)
- Spacing: 8px increments (Tailwind standard)
